# 小学校 学年・教科フォルダ一括作成バッチ

小学校教員向けの授業資料フォルダ構成を一括で作成するWindowsバッチファイルです。

## 概要

このツールは、小学校3〜6年生の学年フォルダと、各学年に対応する教科サブフォルダを自動作成します。教材や授業資料の整理に活用できます。

## 作成されるフォルダ構成

```
├── 3年生/
│   ├── 01国語/
│   ├── 02社会/
│   ├── 03算数/
│   ├── 04理科/
│   ├── 05音楽/
│   ├── 06図工/
│   ├── 07体育/
│   ├── 08英語/
│   ├── 09道徳/
│   ├── 10総合/
│   ├── 11学活/
│   └── 99その他/
├── 4年生/
│   └── （同上）
├── 5年生/
│   └── （同上）
└── 6年生/
    └── （同上）
```

## 使い方

1. `フォルダ作成.bat` をフォルダを作成したい場所に配置
2. ダブルクリックで実行（確認ダイアログが出たら、実行をクリックする）
3. 確認メッセージが表示されたら `y` を入力してEnterキーを押す
4. フォルダが作成される

## 機能

- 実行前の確認プロンプト（誤操作防止）
- 既存フォルダがある場合はスキップ（上書き防止）
- 作成状況のメッセージ表示

## 動作環境

- Windows 10 / 11
- コマンドプロンプト（cmd.exe）

## カスタマイズ

必要に応じてバッチファイルを編集することで、以下の変更が可能です。

- **学年の変更**: `for %%g in (3 4 5 6)` の数字を変更
- **教科の追加・削除**: `for %%s in (...)` 内の教科名を編集
